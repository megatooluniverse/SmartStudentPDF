# SmartStudentPDF
# 🎓 Smart Student PDF Maker

A lightweight, powerful, and completely browser-based tool designed for students to create perfectly formatted A4 PDFs for their assignments and notes. No backend required!

## ✨ Key Features

* **📝 Rich Text Editor:** Fully featured text editor with headings, bullet points, and text alignment.
* **📄 Flawless A4 PDF Export:** Smart pagination ensures text and images never get cut off across pages.
* **🎤 Voice Typing (Speech to Text):** Dictate your notes hands-free. Supports both English and Hindi.
* **🔊 Read Aloud (Text to Speech):** Listen to your typed notes to review them easily.
* **🖼️ Drag & Drop Images:** Seamlessly drop images from your device directly into your assignment.
* **🎨 Custom Design:** Add personalized Watermarks (e.g., "DRAFT") and beautiful page borders.
* **📊 Live Statistics:** Real-time word count, character count, page limit tracker (max 12), and estimated reading time.
* **💾 Auto-Save:** Never lose your work! The app auto-saves your progress locally.
* **🌙 Dark Mode:** Easy on the eyes for late-night study sessions.

## 🛠️ Technologies Used

* **HTML5 & CSS3** (Custom responsive UI)
* **Vanilla JavaScript** (Logic, Speech API, Drag & Drop)
* **[html2pdf.js](https://ekoopmans.github.io/html2pdf.js/)** (For perfect PDF generation & pagination)
* **[FontAwesome](https://fontawesome.com/)** (For beautiful icons)

## 🚀 How to Use / Installation

Since this is a fully static client-side web application, you don't need to install any servers or databases.

1. Clone this repository or download the ZIP file.
2. Extract the folder.
3. Double-click on `index.html` to open it in any modern web browser (Google Chrome or Microsoft Edge recommended for voice features).

## 📝 Important Notes
* **Microphone Access:** The Voice Typing feature requires microphone permissions. Your browser will prompt you to allow access when you click "Start Voice".
* **Page Limit:** To ensure optimal performance and standard assignment sizes, the app warns you if your document exceeds 12 pages.
